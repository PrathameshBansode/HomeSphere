# 🏠 HomeSphere

## 📖 Description

HomeSphere is a backend-driven application built using Spring Boot to manage home-related services efficiently. It follows a clean architecture with proper separation of controllers, services, and repositories.

---

## ✨ Features

* RESTful API structure
* Clean layered architecture
* Maven-based project
* Scalable backend design

---

## 🛠️ Tech Stack

* Java
* Spring Boot
* Maven
* Git & GitHub

---

## ⚙️ Installation Steps

### Clone repo

git clone https://github.com/your-username/HomeSphere.git

### Go to backend

cd backend

### Run project

mvn spring-boot:run

---

## 🚀 Usage

* Run backend
* Use Postman to test APIs
* Server runs on: http://localhost:8080
